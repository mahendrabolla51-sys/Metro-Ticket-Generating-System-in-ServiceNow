 Metro Ticket Generating System (ServiceNow)
 Overview

The Metro Ticket Generating System is a ServiceNow application that automates the metro ticket booking process. Users can book tickets through the Service Catalog, and the system automatically generates a Booking ID, stores the booking details, and creates a QR Code for each ticket.
